## 📦 ملخص المشروع

تم إنشاء **نظام نسخ احتياطي شامل ومتكامل** يدعم:


## 💡 أمثلة الاستخدام

### مثال 1: نسخ بسيط
```bash
python3 backup_advanced.py \
  -s ~/Documents \
  -d /media/external/backup \
  -n my_docs \
  -t tar
```

### مثال 2: نسخ مشفر
```bash
python3 backup_advanced.py \
  -s ~/Private \
  -d /media/external/backup \
  -n private \
  -t encrypted \
  -p "password123"
```

### مثال 3: استخدام تفاعلي
```bash
./backup_system.sh
# ثم اتبع التعليمات على الشاشة
```

## 🔧 للمستخدمين المتقدمين

```bash
# نسخ احتياطي مجدول
crontab -e

# إضافة: نسخ يومي الساعة 2 صباحاً
0 2 * * * python3 /path/to/backup_advanced.py -s ~/data -d /backup -n daily -t tar

# نسخ مع سكريبت مخصص
#!/bin/bash
for dir in ~/Documents ~/Pictures ~/Videos; do
  python3 backup_advanced.py -s "$dir" -d /backup -n "$(basename $dir)" -t zip
done
```
