# موقع مخطط بوابة الخوخة السكنية النموذجية

## نظرة عامة
موقع ويب متكامل لتسويق وبيع قطع الأراضي في مخطط بوابة الخوخة السكنية النموذجية في محافظة الحديدة، اليمن.

## مواصفات المشروع
- **المساحة الإجمالية**: 1,440,000 متر مربع
- **عدد القطع**: حوالي 4,000 قطعة أرض
- **أنواع القطع**:
  - سكنية صغيرة: 160 متر مربع (15%)
  - سكنية متوسطة: 254 متر مربع (70%)
  - تجارية: 300-1600 متر مربع (15%)

## الموقع الجغرافي
- **المديرية**: الخوخة، محافظة الحديدة
- **المسافات**:
  - 5.5 كم من مركز مدينة الخوخة
  - 7 كم من الساحل البحري
  - 20 كم من مدينة حيس
  - 6 كم من الخط الساحلي الرئيسي

## مميزات الموقع

### 🎨 التصميم
- تصميم عصري ومتجاوب مع جميع الأجهزة
- واجهة باللغة العربية مع دعم RTL
- ألوان متدرجة وتأثيرات بصرية جذابة
- خطوط Cairo الأنيقة من Google Fonts

### 📱 الوظائف التفاعلية
- قائمة تنقل متجاوبة للهواتف المحمولة
- تمرير سلس بين الأقسام
- نماذج تفاعلية للاستفسارات
- رسائل تأكيد وتنبيهات
- تأثيرات حركية عند التمرير

### 🏗️ الأقسام الرئيسية
1. **الصفحة الرئيسية**: عرض جذاب مع إحصائيات المشروع
2. **عن المشروع**: تفاصيل شاملة عن المخطط
3. **أنواع القطع**: عرض تفصيلي لكل نوع من القطع
4. **الموقع**: خريطة المسافات والمواقع المهمة
5. **المميزات**: نقاط القوة والمزايا التنافسية
6. **التواصل**: نموذج استفسار وبيانات الاتصال

## الملفات المتضمنة

### 📄 الملفات الأساسية
- **index.html**: الصفحة الرئيسية مع هيكل HTML5 متكامل
- **styles.css**: ملف التنسيقات الرئيسي مع تصميم متجاوب
- **script.js**: ملف JavaScript للتفاعل والوظائف الأساسية
- **config.js**: ملف الإعدادات والتكوين
- **enhancements.js**: وظائف متقدمة إضافية

### 🔧 ملفات التكوين
- **package.json**: إدارة التبعيات وأوامر البناء
- **manifest.json**: إعدادات تطبيق الويب التقدمي (PWA)
- **robots.txt**: توجيهات محركات البحث
- **sitemap.xml**: خريطة الموقع لمحركات البحث
- **.gitignore**: ملفات Git المستبعدة

### 🚀 ملفات التشغيل
- **start.sh**: سكريبت تشغيل سريع لـ Linux/Mac
- **start.bat**: سكريبت تشغيل سريع لـ Windows
- **README.md**: دليل الاستخدام الشامل

### 📁 المجلدات المقترحة
- **images/**: الصور والأيقونات
- **dist/**: الملفات المحسنة للنشر
- **docs/**: الوثائق الإضافية

## كيفية الاستخدام

### 1. التشغيل السريع

#### على نظام Linux/Mac:
```bash
# اجعل الملف قابل للتنفيذ (مرة واحدة فقط)
chmod +x start.sh

# تشغيل الموقع
./start.sh
```

#### على نظام Windows:
```cmd
# تشغيل الموقع
start.bat
```

### 2. التشغيل اليدوي

#### تثبيت التبعيات:
```bash
npm install
```

#### تشغيل الخادم المحلي:
```bash
# باستخدام live-server
npm start

# أو باستخدام http-server
npm run serve

# أو فتح الملف مباشرة
open index.html
```

### 3. البناء والنشر
```bash
# بناء الملفات المحسنة
npm run build

# تحسين الصور
npm run optimize-images

# النشر الكامل
npm run deploy
```

### 4. التخصيص
- عدّل الإعدادات في ملف `config.js`
- غيّر بيانات الاتصال والأسعار
- أضف الصور الحقيقية في مجلد `images/`
- اربط النماذج بقاعدة البيانات
