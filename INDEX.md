📖 اقرأ: **OVERVIEW.md** ثم **INSTALLATION_GUIDE.md**

### أريد تفاصيل كل شيء:
📖 اقرأ: **BACKUP_README.md**

### أريد تجربة النظام:
```bash
chmod +x test_backup.sh && ./test_backup.sh
```

### أريد نسخ احتياطي سريع الآن:
```bash
python3 backup_advanced.py -s ~/Documents -d /media/backup -n docs -t tar
```
